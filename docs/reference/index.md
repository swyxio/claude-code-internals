## Search by exact identifier

The site search indexes headings, claim IDs, anchor IDs, flags, environment
variables, protocol methods, and reconstructed filenames. Exact searches such
as `permissions.disable-bypass`, `tools/call`, `--allowedTools`, or
`CLAUDE_CODE_ENTRYPOINT` lead to both explanation and source-of-truth links.
