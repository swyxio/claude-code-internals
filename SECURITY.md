# Security policy

This repository documents security controls and trust boundaries. It is not an
appropriate place to publish a working exploit against Claude Code or
Anthropic services.

For a suspected Anthropic vulnerability, follow Anthropic's
[Responsible Disclosure Policy](https://www.anthropic.com/responsible-disclosure-policy).
Keep technical details private until coordinated disclosure is authorized.

For a vulnerability in this repository's parser or documentation build, open a
GitHub security advisory against `swyxio/claude-code-internals` if available,
or contact the repository owner privately.

Never attach executable fixtures, credentials, user transcripts, or private
configuration to a public issue.
